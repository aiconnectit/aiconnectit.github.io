html语言的构成

lang=”en” 告诉搜索引擎爬虫，我们的网站是关于什么内容的，这里的意思是告诉搜索引擎是英文内容的网站，利于抓取

属性值必须用双引号引用起来

style=”风格设计”

a标签的作用（超链接）

   <a href="https://www.baidu.com（给浏览器用的）"  target="_blank">百度一下（给人用的） 
 
   这里面的内容可以是任意形式
   图片
   CSS
   文字
   这些都是可以放到这里里面的

   <\a>href ---->hyperText reference(超文本引用的意思，添加的是你指向的所在地址)

还可以放 “target” 是目标的意思，添加这个标识符就可以设置超文本链接在新的标签页中打开，如果你不想在新的标签页中打开就可以不用添加。

target=”_blank”等号后面的双引号添加的值是一个空值

a标签的作用
1、超链接
2、锚点 id=deng <a href=”#deng”
3、打电话 给谁打电话
4、协议限定符 可以添加javascript代码

“a”是英文anchor的英文缩写 意思是锚

style意思是类型风格，可以设置类型风格

form 表单标签

可以发送数据，可以将前端数据发送给后端数据

  <form method="get" action="" > 

   <input: type="text" name="username" value=""> 账号
   <input: type="password" name="password" value=""> 密码
   
   <input: type="submit"> 提交数据（提交到后端数据）

   <input: type="radio" name="将单选框统一到同一道选择题" value="数据值"> 单选框（必须统一到同一道题用name）

提交的数据必须要有数据名和数据值
username 这个数据名（用户在这个框子里面填写的名字就是数据值）
password 这个数据名（用户在这个框子里面填写的密码就是数据值）
<\form>

method=”get” 获取的数据

action=”” 发送给谁的地址

value=”” 属性值

复选框（checkbox）

把type=”checkbox”的值改成checkbox就可以变成复选框

默认选中给用户提供很大的体验

checked=”checked”默认选中
